# 拼豆网站需求文档

## 项目概述
一个纯前端的拼豆（Perler Beads）模版生成工具，用户上传照片后可生成像字帖一样的拼豆模版。

## 核心功能

### 1. 图片上传与处理
- 支持用户上传照片
- 提供裁剪功能
- 支持调整图片大小
- 使用Canvas API进行纯前端处理

### 2. 拼豆规格
- 标准豆（5mm）

### 3. 模版尺寸
- 根据原图自动推荐模版尺寸
- 用户可手动调整行列数

### 4. 品牌色系
- 预设多个品牌色系（Hama、Perler、Artkal等）
- 保留完整调色板让用户选择
- 用户可以选择不同品牌的色系

### 5. 预览功能
- 生成前让用户预览效果
- 显示建议颜色数

### 6. 模版输出
#### 输出内容
- 显示每个位置的豆子及颜色
- 显示行列编号
- 统计每种颜色的豆子数量

#### 输出方式
- 支持下载为图片文件（PNG）
- 支持打印预览和打印

#### 分页处理
- 大图自动分页（按A4纸尺寸）
- 显示拼接标识方便用户拼接

### 7. 本地保存
- 用户记录保存到localStorage

## 技术栈
- Vue 3
- Canvas API（纯前端处理）
- localStorage

## 界面风格
- 简约可爱
